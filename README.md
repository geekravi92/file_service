## Description

File Service

## Installation

```bash
$ npm install
```

## Put The Data File
```bash
file_service/data/<data-excel-file.xlsx>
```

## Running the app

```bash
# development
$ FILENAME=<data-excel-file.xlsx> npm run start

# watch mode
$ FILENAME=<data-excel-file.xlsx> npm run start:dev
```
